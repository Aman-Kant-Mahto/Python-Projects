# 📩 Python Automation: WhatsApp, SMS & Email

This project demonstrates how to automate sending **WhatsApp messages, SMS, and Emails** using Python.

🔗 Full write-up: [Medium Blog](https://medium.com/@amanmahto669/sending-whatsapp-message-sms-email-using-python-97979c77933c)

---

## 📌 Features
- ✅ Send **WhatsApp messages** using [PyWhatKit](https://pypi.org/project/pywhatkit/)  
- ✅ Send **SMS messages** using [Twilio](https://www.twilio.com/sms)  
- ✅ Send **Emails** using Python’s built-in `smtplib` and Gmail SMTP  

---

## 📂 Files
- `whatsapp.py` → Send a WhatsApp message via WhatsApp Web  
- `sms.py` → Send an SMS using Twilio API  
- `email.py` → Send an email through Gmail SMTP
